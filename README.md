Customer Churn Analysis Dashboard (Task 2)

## Overview

This project analyzes customer churn data to identify key patterns, retention drivers, and factors influencing customer behavior in a subscription-based business. The goal is to generate actionable insights that support data-driven decision-making.

## Dataset

* Source: Telco Customer Churn Dataset
* Contains information on:

  * Customer demographics
  * Contract types
  * Payment methods
  * Services used
  * Tenure and billing details
  * Churn status (Yes/No)

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning and Transformation

## Dashboard Components

### 1. Churn by Contract Type

* Month-to-month contracts show the highest churn
* Long-term contracts (1-year, 2-year) have better retention

### 2. Churn by Payment Method

* Electronic check users have the highest churn
* Automatic payments reduce churn significantly

### 3. Churn by Tenure Group

* New customers (0–12 months) churn the most
* Retention improves as tenure increases

### 4. Churn by Monthly Charges

* Higher monthly charges are strongly linked to higher churn
* Lower-cost plans retain more customers

## Key Insights

* Month-to-month customers have the highest churn
* Electronic check users churn the most
* Customers with low tenure are more likely to leave
* Higher monthly charges increase churn risk

## Recommendations

* Encourage long-term contracts through incentives
* Improve onboarding experience for new customers
* Promote automatic payment methods
* Review pricing strategies for expensive plans

## Key Metrics

* Total Customers: 7043
* Churned Customers: 1869
* Churn Rate: 26.5%
